# TreeDrawer - 目录树生成器

一个简单易用的GUI工具，用于生成目录的树状图，适合用于文档编写和项目说明。

## 功能特点

- 🌳 生成清晰的目录树状图
- 📁 可选择是否显示文件（只显示文件夹或包含文件）
- 👁️ 可选择是否忽略隐藏文件和文件夹
- 📏 支持限制最大显示深度
- 📋 一键复制到剪贴板
- 🖥️ 友好的图形界面

## 使用方法

### 启动程序

```bash
# 在TreeDrawer文件夹中运行
python main.py
```

或者直接运行GUI文件：

```bash
python tree_gui.py
```

### 操作步骤

1. **选择目录**: 点击"浏览"按钮选择要生成树状图的目录，或直接在路径输入框中输入路径
2. **设置选项**:
   - 显示文件: 勾选则显示文件名，不勾选则只显示文件夹
   - 忽略隐藏文件: 勾选则忽略以`.`开头的文件和文件夹
   - 最大深度: 限制显示的目录层数（空白表示不限制）
3. **生成树状图**: 点击"生成目录树"按钮
4. **复制结果**: 点击"复制到剪贴板"按钮，即可粘贴到README或其他文档中

## 文件说明

- `main.py`: 程序入口文件
- `tree_gui.py`: GUI界面主程序
- `tree_generator.py`: 目录树生成核心逻辑
- `README.md`: 使用说明文档

## 系统要求

- Python 3.6+
- tkinter (通常Python自带)
- 支持Windows、macOS、Linux

## 注意事项

- 对于大型目录，建议设置适当的最大深度以避免输出过多内容
- 程序会自动处理权限错误，无法访问的目录会显示相应提示
- 生成的树状图使用UTF-8编码，确保在支持Unicode的编辑器中查看
